認証関連
─────────────────────────────────────────────
POST   /auth/login
       - Body: { email, password }
       - 認証: 不要
       - 説明: ログインしアクセストークン・リフレッシュトークン発行

POST   /auth/signup
       - Body: { username, email, password }
       - 認証: 不要
       - 説明: 新規ユーザー登録

POST   /auth/logout
       - Cookie: refresh_token (HttpOnly)
       - 認証: 必須 (クッキー送信)
       - 説明: リフレッシュトークンをDBから削除しクッキー削除

POST   /auth/refresh
       - Cookie: refresh_token (HttpOnly)
       - 認証: 不要（クッキーで認証）
       - 説明: アクセストークン再発行

ユーザー関連 /u/[userId]

GET    /users/:userId
       - 認証: 必須
       - 説明: ユーザープロフィール取得

       - 認証: 必須

GET    /users/:userId/cards
       - 認証: 必須
       - 説明: ユーザーの全カード一覧取得
